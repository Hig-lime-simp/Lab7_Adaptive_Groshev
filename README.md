## Часть 1: Изучение Flexbox

### Изученные свойства:
- `display: flex` - создание flex-контейнера
- `flex-direction` - направление элементов (row, column, row-reverse, column-reverse)
- `justify-content` - выравнивание по главной оси
- `align-items` - выравнивание по поперечной оси
- `flex-wrap` - перенос элементов
- `gap` - отступы между элементами
- `flex-grow`, `flex-shrink`, `flex-basis` - управление размерами

---

## Часть 2: Изучение CSS Grid

### Изученные свойства:
- `display: grid` - создание grid-контейнера
- `grid-template-columns` - определение столбцов
- `grid-template-rows` - определение строк
- `grid-template-areas` - именованные области
- `gap` - отступы между ячейками
- `fr` (fraction) - единица измерения для распределения пространства
- `repeat()` - функция для повторения значений

### Практические примеры:
1. Простая сетка 3x3
2. Макет страницы с именованными областями
3. Адаптивная галерея изображений

## Часть 3: Медиа-запросы и адаптивность

#### Контрольные точки (breakpoints):
- Мобильные: < 768px
- Планшеты: 768px - 1024px
- Десктопы: ≥ 1024px

### Mobile First подход:
Разработка начинается с мобильной версии, затем добавляются стили для больших экранов.
